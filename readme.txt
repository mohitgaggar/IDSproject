1.Run Inserting_Nan.py(python Inserting_Nan.py) present in Initial folder to insert some Nan values (to make ir real world like)

3.Run data_standardization.py present in Standardization folder to Standardize data
4.Run data_normalization.py present in Standardization folder to Normalize data
5.Run scatte.ipynb file for all plots ,chi square test
